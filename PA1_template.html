Imputing missing values

  1. Calculating and reporting the total number of missing values in the dataset
missing <- is.na(rawData$steps)
nMissing <- summary(missing)[3]

There are 2304 missing values.

  1. Devising a strategy for filling in all of the missing values in the dataset.

All of the missing values are filled in with mean value for that 5-minute interval.

  1. Creating a new dataset that is equal to the original dataset but with the missing data filled in.

Here is the code to fill it.

fill.value <- function(steps, interval) {
    filled <- NA
    if (!is.na(steps)) 
        filled <- c(steps) else filled <- (averages[averages$interval == interval, "steps"])
    return(filled)
}
filled.data <- rawData
filled.data$steps <- mapply(fill.value, filled.data$steps, filled.data$interval)
